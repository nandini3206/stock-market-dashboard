# 📈 Stock Market Dashboard & Analyzer

An AI-powered stock analysis dashboard built using Python, Streamlit, yfinance, and Groq AI.

---

## 🚀 Live Demo
https://stock-market-dashboard-manb.streamlit.app

---

## 💻 GitHub Repository
https://github.com/nandini3206/stock-market-dashboard

---

## 🔥 Features

- Real-time stock market data using yfinance
- Moving Average Analysis (20-day & 50-day)
- Buy/Sell Trading Signals
- Risk Analysis
- AI Finance Chatbot using Groq + LLaMA 3
- Candlestick Chart
- Stock Comparison
- Trading Volume Analysis
- Currency Toggle (USD / INR)
- Interactive Streamlit Dashboard

## ▶️ Run Locally

### 1️⃣ Clone the repository

```bash
git clone https://github.com/nandini3206/stock-market-dashboard.git
```

### 2️⃣ Move into project folder

```bash
cd stock-market-dashboard
```

### 3️⃣ Create virtual environment

```bash
python -m venv venv
```

### 4️⃣ Activate virtual environment

#### Windows

```bash
venv\Scripts\activate
```

#### Mac/Linux

```bash
source venv/bin/activate
```

### 5️⃣ Install dependencies

```bash
pip install -r requirements.txt
```

### 6️⃣ Create `.env` file

```env
GROQ_API_KEY=your_groq_api_key_here
```

Get Groq API key from:
https://console.groq.com

### 7️⃣ Run the Streamlit app

```bash
streamlit run app.py
```
